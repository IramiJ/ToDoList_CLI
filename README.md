# Multi Tool CLI

A command-line application written in **Python** that combines multiple small tools in one program.

This project was created to practice basic programming logic and working with user input in the terminal.

---

## Features

### To-Do List
- Add tasks
- Delete tasks
- View all tasks
- Assign a due time to tasks

### File Organizer
- Sort files in a directory by file type
- Automatically creates folders (Images, Texts, PDFs, Others)

### Calculator
- Basic arithmetic operations:
  - add
  - subtract
  - multiply
  - divide

---

## Tech Stack

- Python 3

---

## Run

```bash
python multi_tool_cli.py
```
